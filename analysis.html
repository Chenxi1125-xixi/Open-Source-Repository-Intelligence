Analysis

Interactive views of repository behavior

This page brings together the project's analytical layer: engagement, activity, quality, and concentration. The goal is not just to describe repositories, but to explain which signals are most closely associated with visibility and sustained adoption.

Data Scope 14,644 repositories

All charts on this page are driven by the cleaned metadata file derived from the full repository source.

Signal Strength Stars and forks are tightly linked

Across the full dataset, stars and forks show a very strong relationship, while stars and commits are much weaker.

Documentation Effect README presence matters

Repositories with a README materially outperform those without one on both visibility and development activity.
